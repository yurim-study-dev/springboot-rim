## 📖 목차

1️⃣ **스프링 프레임워크 핵심 원리**
- IoC / DI 개념
- 스프링 컨테이너와 빈(Bean)
- 싱글톤 패턴과 스프링

2️⃣ **스프링 부트 시작하기**
- 스프링 부트 프로젝트 생성
- 자동 설정(Auto Configuration)
- 내장 톰캣과 실행 방법

3️⃣ **Spring MVC 웹 개발 기초**
- MVC 패턴 이해
- Controller / Service / Repository 계층 구조
- Thymeleaf 템플릿 엔진

4️⃣ **Spring JDBC 데이터 접근**
- JdbcTemplate 사용법
- 트랜잭션 관리 (`@Transactional`)
- DataSource 설정

5️⃣ **Spring MVC 웹 개발 심화**
- 검증(Validation)과 Bean Validation
- 로그인 / 세션 처리
- 필터(Filter)와 인터셉터(Interceptor)

6️⃣ **Spring Data JPA**
- JPA 기본 개념 (ORM)
- `@Entity`, `@Repository`, `JpaRepository`
- JPQL과 QueryDSL

7️⃣ **Spring REST API**
- RESTful API 설계 원칙
- `@RestController`, `@RequestBody`, `@ResponseBody`
- 예외 처리 (`@ExceptionHandler`, `@ControllerAdvice`)

8️⃣ **Spring Security**
- 인증(Authentication)과 인가(Authorization)
- Security Filter Chain
- JWT 기반 인증

9️⃣ **Spring Batch**
- 배치 처리 개념
- Job / Step / Tasklet 구조
- 스케줄러 연동
